## Scripts Description

### Script 1 — System Identity Report
Displays system information including Linux distribution, kernel version, current user, home directory, uptime, date, and license information.

### Script 2 — FOSS Package Inspector
Checks whether Git is installed on the system, displays its version, and provides a brief description using a case statement.

### Script 3 — Disk and Permission Auditor
Analyzes key system directories and reports their size, ownership, and permissions using a loop.

### Script 4 — Log File Analyzer
Reads a log file, counts occurrences of a given keyword (default: error), and displays the last five matching entries.

### Script 5 — Open Source Manifesto Generator
Takes user input and generates a personalized open-source manifesto, saving it to a text file.
